## 🚀 Як запустити

```bash
# Клонувати → скопіювати env → піднести Docker → поставити Drupal
git clone <repo-url> bearcom && cd bearcom
cp .env.example .env
docker compose up -d --build
docker compose exec php bash scripts/install.sh

# Сайт:  http://localhost
# Адмін: http://localhost/user/login  (admin / admin)
# Пошта: http://localhost:8025
```

### Щоденні команди

```bash
drush cr          # очистити кеш
drush cex         # експортувати конфіг (після змін в адмінці)
drush cim         # імпортувати конфіг (після пулу)
drush uli         # одноразовий логін
```

*(виконувати в PHP-контейнері: `docker compose exec php bash`, потім `cd web`)*
